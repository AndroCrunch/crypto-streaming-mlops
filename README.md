# Crypto Streaming MLOps Pipeline

An end-to-end machine learning engineering project focused on building a production-style MLOps pipeline using real-world cryptocurrency market data from Binance.

## Project Overview

The purpose of this project is not simply to build a predictive model, but to design and implement a complete machine learning lifecycle pipeline that simulates a real-world production environment.

The project covers every major stage of a modern machine learning system, including data ingestion, feature engineering, model training, experiment tracking, deployment, real-time inference, monitoring, and continuous data streaming.

Cryptocurrency market data is used as the underlying data source because it provides both historical datasets for offline model development and live streaming market data for real-time production simulation.

The machine learning model itself is only one component of the larger system architecture.

## Core Objective

Build a complete end-to-end ML pipeline that demonstrates how machine learning systems are developed, deployed, monitored, and maintained in production environments.

The project focuses on machine learning engineering and MLOps best practices rather than only maximizing predictive accuracy.
